# Change Log

All notable changes to the "LazyPredict UI" extension will be documented in this file.

## [0.0.1] - 2025-05-19

### Added
- Initial release of LazyPredict UI extension
- Main webview panel with React UI components
- Dataset upload and selection functionality
- Model selection for classification and regression
- Results display with model performance metrics
- Python environment detection and LazyPredict integration
- Basic error handling and user feedback

### Known Issues
- Extension requires LazyPredict to be pre-installed in Python environment
- Limited support for large datasets
